ИНСТРУКЦИЯ ПО ЗАПУСКУ ПРИЛОЖЕНИЯ БЕЗ СРЕДЫ РАЗРАБОТКИ. Если у вас она стоит, то просто добавьте проект с гитхаба.

1. Установите java jdk (https://www.oracle.com/java/technologies/downloads/#jdk17-windows)

2. Откройте powershell от имени администратора и введите туда эту команду:
[Environment]::SetEnvironmentVariable("JAVA_HOME", "javapath", "Machine")
 заменив "javapath" на путь до jdk, который только что установили.

3. Введите команды 
$OLDPATH = [System.Environment]::GetEnvironmentVariable('PATH','machine')
$JAVAPATH = [System.Environment]::GetEnvironmentVariable('JAVA_HOME','machine')
$NEWPATH = "$OLDPATH;$JAVAPATH\bin"
[Environment]::SetEnvironmentVariable("PATH", "$NEWPATH", "Machine")

4. Перезагрузите компьютер

5.Загрузите установщик ApacheTomcat(https://tomcat.apache.org/download-90.cgi)

6. Установите его, указав путь на jdk там, где потребуется. Обязательно поставьте галочку на hostManager.

7.Зайдите в директорию, в которую установили TomCat, в папку webApps.

8. Cкачайте файл ROOT.war из директории проект в GitHub

9. Удалите папку ROOT,  вставьте скачанный файл в эту директорию.

10. Запустите TomCat, зайдя в папку TomCat/bin и запустив  startup.bat

11. Остановите сервер, закрыв окно с консолью.

12. Скачайте файл 1.mv.db из директории этого проекта на GitHub finalProject/finalProject/src/main/resources/static/
и вставьте с заменой файлов в папку с Tomcat bin\src\main\resources\static

13. Все, можно пользоваться
